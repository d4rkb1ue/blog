Sublime Select Tricks

18 Feb 2017 | tags: [ Tools ]

问题

在选择的时候遇到了一些问题，如下面的代码

if (max && p.compareTo(ret) > 0) ret = p;

我在选择变量p的时候，shift选中p然后Command + D，后果就是同时选中了compareTo中的p。

解决方案1

选中p之后，Command + D选中compareTo中的p，此时Command + K，Command + D可以取消选择上个选中的p，即compareTo中的p，并选中下一个p。

不过这个方法很麻烦。其实 Sublime 是可以识别变量的。

解决方案2

不要选中p！ 不要选中p！ 不要选中p！

直接将光标放在p之前或之后，Command + D即可自动选择变量。

  • 还有一个关于选择的快捷键，Command + U 可以撤销最后一个选择。

其他快捷键

  • 选择全部 token Ctrl + Command + G

  • HTML 中选中一对标签（需要 Emmet） Command + Shift + K

Reference
